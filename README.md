# Jayce's Lab

Minimal technical blog hosted on GitHub Pages and powered by Jekyll.

- Website: <https://jayceto946-byte.github.io>
- Online writing: <https://jayceto946-byte.github.io/write/>

## Publish a post

Create a Markdown file under `_posts` using `YYYY-MM-DD-short-english-title.md`.

```yaml
---
layout: post
title: "文章标题"
date: 2026-07-19 12:00:00 +0800
description: "一句话摘要"
---
```

Commit the file to `main`. GitHub Pages will build and publish it automatically.
