在线写作

登录 GitHub 后，可以直接创建 Markdown 文章或上传文件。提交后博客会自动更新。

新建文章 上传 Markdown 上传图片

发布步骤

  1. 点击“新建文章”，把文件名中的 article-title 改成简短英文标题。
  2. 修改文章标题，然后用 Markdown 写正文。
  3. 点击 Commit changes，选择直接提交到 main。
  4. 等待约一分钟，文章会自动出现在首页。

“写作”页面可以公开访问，但只有拥有仓库写权限并登录 GitHub 的账号才能真正发布。不要在文章或仓库中填写密码、Token 等敏感信息。
